要在本机上运行数据库初始化批处理文件必须要进行：
1.sqlplus参数配置
根据本机登录名和登录口令，还有Oracle服务名进行修改参数

对以上参数修改批处理中：
sqlplus system/62591102@127.0.0.1:1521/orcl 

2.java jar批处理参数配置
第一个参数 Oracle服务名
第二个参数 登录名
第三个参数 登录口令

对以上参数修改批处理中的：
java -jar xxx.jar orcl system 62591102

初始化数据库中运行jar包和运行客户端jar包都要进行修改！
